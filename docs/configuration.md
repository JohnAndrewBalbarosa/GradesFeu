# Configuration

Set values in `.env`.

## Core URLs

- `GRADES_URL`
- `CURRICULUM_URL`
- `OFFERINGS_URL`

## Cache Paths

- `CACHE_FILE`
- `AVAILABILITY_CACHE_FILE`

## Analysis Thresholds

- `HONOR_SUMMA_MIN`, `HONOR_MAGNA_MIN`, `HONOR_CUM_MIN`
- `HONOR_SUMMA_MAX`, `HONOR_MAGNA_MAX`, `HONOR_CUM_MAX`
- `OUTLIER_METHOD`, `OUTLIER_TAIL`, `OUTLIER_Z_THRESHOLD`
- `BASELINE_ESTIMATOR`
- `FAILED_GRADE_VALUES`
